## Algorithms and Methods

### Variational Quantum Eigensolver (VQE)
The VQE is a hybrid quantum-classical algorithm used to find the minimum eigenvalue (ground state energy) of a Hamiltonian $H$.

**Pseudocode:**
1. **Decomposition**: Represent $H$ as a sum of Pauli strings: $H = \sum_i c_i P_i$.
2. **Ansatz Selection**: Define a parameterized quantum circuit $U(\theta)$ to prepare a state $|\psi(\theta)\rangle = U(\theta)|0\rangle$.
3. **Execution Loop**:
    a. Measure expectation values $\langle \psi(\theta) | P_i | \psi(\theta) \rangle$ for each Pauli string.
    b. Compute weighted sum: $E(\theta) = \sum_i c_i \langle P_i \rangle_\theta$.
    c. Use a classical optimizer (e.g., BFGS) to update $\theta$ to minimize $E$.
4. **Convergence**: Stop when $E(\theta)$ stabilizes at the minimum.

### Mapping Quasispin to Qubits (Lipkin Model)
For $J=1$ (3 states), we map the system to a 2-qubit subspace ($2^2=4$ states). The quasispin operators $J_z, J_+, J_-$ are represented as sums of Pauli matrices (e.g., $J_z = \frac{1}{2}(Z_1 + Z_2)$).

## Parts Overview

### Part (a): Quantum Basics
- Basis implementation ($|0\rangle, |1\rangle$).
- Pauli matrix application.
- Bell state generation.
- Entanglement entropy calculation.

### Part (b): One-Qubit Hamiltonian
- $H(\lambda) = H_0 + \lambda H_I$.
- Pauli decomposition.
- Exact eigenvalue sweep.

### Part (c): VQE for One-Qubit Hamiltonian
- Implementation of a simple $R_y$ ansatz.
- Optimization using gradient-based solvers.
- Comparison with exact results.

### Part (d): Two-Qubit Hamiltonian
- Interacting two-qubit system.
- Exact diagonalization and level crossing analysis.
- Von Neumann entropy as a measure of entanglement.

### Part (e): VQE for Two-Qubit Hamiltonian
- Hardware-efficient ansatz ($R_y$-CNOT-$R_y$).
- Optimization and accuracy benchmarking.

### Part (f): Lipkin Model
- Second quantization Hamiltonian construction using quasispin operators.
- **Pauli decomposition**: J=1 mapped to `(ε/2)(Z₁+Z₂) + (V/2)(X₁X₂ − Y₁Y₂)`, verified by projection to triplet subspace.
- **Pauli decomposition**: J=2 mapped to `(ε/2)Σᵢ Zᵢ + (V/2)Σᵢ<ⱼ(XᵢXⱼ − YᵢYⱼ)`, J=2 sector eigenvalues verified.
- Classical diagonalization for $J=1$ and $J=2$ with Hartree-Fock comparison.

### Part (g): VQE for Lipkin Model
- Mapping quasispin to Pauli strings for N-qubit simulation.
- VQE optimization for the Lipkin model ground state with hardware-efficient ansatz.
- Triplet-sector verification: VQE ground state confirmed in maximal-J sector.
- Cross-validation against quasispin exact diagonalization.
